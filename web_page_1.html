Array Data Structure

Arrays are one of most important topic to conquer DSA. They are used to store multiple items of same data type.
For Eg. If you have to store marks of 100 students in a class than creating 100 different variables will make the program look messy and also it is difficult to debug it so instead of this we define array.

One of the most important point to note is that in array elements are assigned the contiginous memory block which you can understand more in given example.

syntax to declare Array:

Data-type name_of_array[size];

syntax to initialize Array:
               Data-type name[size]={element1,element2,....up to the size of the array};

  1. To Traverse a given array
