Understanding Time Complexity

In computer science, the time complexity is the computational complexity that describes the amount
of computer time it takes to run an algorithm. We donate it with Big Oh notation O(1) (readed as Big
Oh of time complexity 1).

    • Understanding Space Complexity

    Space Complexity of an algorithm is the total space taken by the algorithm with respect to
    the input size. Space complexity includes both Auxiliary space and space used by input.

     
              int addSequence (int n){
                int sum = 0;
                for (int i = 0; i < n; i++){
                sum += pairSum(i, i+1);
                    }
                return sum;
                }
     
              int pairSum(int x, int y){
                return x + y;
                  }
              There will be roughly O(n) calls to pairSum. However, those
              calls do not exist simultaneously on the call stack,
              so you only need O(1) space.
